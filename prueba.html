Introducción al ESP32

El ESP32 es un microcontrolador con WiFi y Bluetooth.

Fue desarrollado por Espressif Systems como sucesor del ESP8266.

Cuenta con múltiples interfaces: SPI, I2C, UART, ADC y más.

Puede ser programado con Arduino IDE, PlatformIO y ESP-IDF.

Soporta conexiones WiFi y BLE para aplicaciones IoT.

Incluye seguridad criptográfica en hardware.

Es ideal para proyectos de bajo consumo.

Puede utilizarse con baterías de litio de 3.7V.

Tiene múltiples variantes, como el ESP32-WROOM-32.

Es ampliamente usado en la comunidad maker.

ESP32.begin(); Serial.begin(115200); pinMode(2, OUTPUT); digitalWrite(2, HIGH); delay(1000);
  • WiFi integrado
  • Bluetooth BLE
  • Doble núcleo
  • GPIO multipropósito
  • Comunidad activa

Configuración inicial

Se recomienda usar Arduino IDE o PlatformIO.

Debes instalar los drivers USB del ESP32.

En Arduino IDE, añade el soporte de ESP32 desde el Gestor de Placas.

Ve a Preferencias y agrega:

https://dl.espressif.com/dl/package_esp32_index.json

Después, abre el Gestor de Placas e instala "esp32 by Espressif Systems".

Selecciona la placa "ESP32 Dev Module".

Conecta el ESP32 y selecciona el puerto adecuado.

Haz clic en subir para cargar tu primer programa.

¡Listo para comenzar!

void setup() { Serial.begin(115200); } void loop() { Serial.println("Hola ESP32"); }

Primer programa

Vamos a encender y apagar un LED interno.

Este ejercicio permite verificar la instalación y configuración.

El pin GPIO2 suele tener un LED conectado en muchas placas.

El programa usa las funciones pinMode y digitalWrite.

Usamos delay para alternar entre encendido y apagado.

El código se ejecuta indefinidamente dentro de loop().

Puedes modificar el tiempo para cambiar la frecuencia del parpadeo.

Este proyecto es una base para probar relés, sensores y más.

Te permite entender cómo se controlan los pines del ESP32.

¡Es el clásico "Hola Mundo" del hardware!

void setup() { pinMode(2, OUTPUT); } void loop() { digitalWrite(2, HIGH); delay(1000); digitalWrite(2, LOW); delay(1000); } // LED parpadeando
  • GPIO2 usado por defecto
  • Parpadeo controlado con delay
  • Programa se repite en loop()
  • Verificación inicial del sistema
  • Puede modificarse para pruebas

Aplicaciones prácticas

El ESP32 puede usarse en monitoreo ambiental.

También en automatización del hogar.

Se puede conectar a servicios en la nube.

Control de motores o relés es muy común.

Los sensores más usados son DHT11, MQ-135, y ultrasonido.

Se puede alimentar con baterías recargables.

Existen aplicaciones de visualización con Blynk o Node-RED.

También puede ser servidor web embebido.

Admite programación con MicroPython y JavaScript (Espruino).

Hay kits de desarrollo con pantalla OLED y sensores incluidos.

#include <WiFi.h> WiFi.begin("tuSSID", "tuPassword"); while (!WiFi.isConnected()) { delay(500); } Serial.println(WiFi.localIP()); // Conexión WiFi básica
